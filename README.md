# team-time

Given a list of arrays that represent when a Person is busy, determine the overall free times for the group of individuals.

## Example

To Run:
```bash
> python main.py
```

Input:
```python
input =
[[0, 4], [5, 9], [10, 14]], # person 1
[[6, 9], [15, 17]], # person 2
[[3, 9]] # person 2
```

Output:
```bash
[[9, 10], [14, 15], [17, 23]]
```
