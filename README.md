# team-time

From input file, create a list of arrays that represent when each person in the group is busy and determine the overall free times for the group.

## Example

To Run:
```bash
> python main.py
```

Input as list of arrays:
```python
input =
[[0, 4], [5, 9], [10, 14]], # person 1
[[6, 9], [15, 17]], # person 2
[[3, 9]] # person 3
```

Output:
```bash
[[9, 10], [14, 15], [17, 23]]
```
